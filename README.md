# Airport Management System

A specialized database management system designed to handle complex airport operations and flight logistics.

# Core Features

  Flight Scheduling: Manage departures, arrivals, and real-time flight status.

  Passenger Records: Secure storage for passenger information and booking history.

  Asset Management: Track aircraft types, maintenance schedules, and hangar assignments.

  Staff Coordination: Organize pilot, cabin crew, and ground staff shifts.
  
# Database Architecture

The system is built on a relational model ensuring data integrity across:

  Flights Table: Links routes, times, and aircraft.

  Passengers Table: Tracks individual travel logs.

  Personnel Table: Role-based access and scheduling.

# Tech Stack

  SQL / T-SQL - Relational database design and stored procedures.

  C# / .NET - Backend integration logic.

  Entity Framework - Object-Relational Mapping (ORM).
